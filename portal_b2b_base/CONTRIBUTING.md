# Guía de Contribución - Portal B2B Base

¡Gracias por tu interés en contribuir a Portal B2B Base! Este documento proporciona directrices y procedimientos para contribuir.

## Código de Conducta

Este proyecto y todos los participantes están regidos por nuestro [Código de Conducta](CODE_OF_CONDUCT.md). Al participar, se espera que mantengas este código.

## ¿Cómo Puedo Contribuir?

### Reportar Bugs

Antes de crear un reporte de bug, verifica la lista de issues ya que podrías encontrar que no necesitas crear uno. Cuando crees un reporte de bug, incluye:

- **Título descriptivo**
- **Descripción clara del problema**
- **Pasos para reproducir**
- **Comportamiento observado**
- **Comportamiento esperado**
- **Capturas de pantalla** (si es aplicable)
- **Versión de Odoo**
- **Sistema operativo**

### Sugerir Mejoras

Las sugerencias de mejora son bienvenidas. Cuando crees una sugerencia de mejora, incluye:

- **Título descriptivo**
- **Descripción clara de la mejora**
- **Justificación de por qué sería útil**
- **Ejemplos de cómo funcionaría**

### Pull Requests

- Sigue el [Estilo de Código](#estilo-de-código)
- Incluye pruebas apropiadas
- Actualiza la documentación
- Termina todos los archivos con una nueva línea
- Evita cambios de plataforma en los diffs

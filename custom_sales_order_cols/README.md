# Sale Order Paid Column

Módulo para Odoo 17 Community que añade una columna de estado "Pagado" en los pedidos de venta.

## Características:
- Campo booleano "Pagado" en pedidos de venta
- Editable directamente desde la vista de lista
- Botón para marcar/desmarcar como pagado
- Registro automático en chatter con fecha y usuario

## Instalación:
1. Copiar el módulo en addons/
2. Actualizar lista de módulos
3. Instalar 'Sale Order Paid Column'

## Uso:
- Marcar pedidos como pagados desde la vista de lista o formulario
- Ver historial de cambios en el chatter del pedido
